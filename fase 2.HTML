Resumen de Etiquetas HTML

1. Etiquetas Básicas

Nombre Inicio Cierre Descripción
html <html> </html> Contenedor principal del documento HTML.
head <head> </head> Contiene metadatos del documento.
body <body> </body> Contiene el contenido visible de la página.
title <title> </title> Título de la página que aparece en la pestaña del navegador.
p <p> </p> Define un párrafo de texto.

Ejemplo de etiqueta básica:

Este es un párrafo utilizando la etiqueta <p>.

2. Etiquetas de Tablas

Nombre Inicio Cierre Descripción
table <table> </table> Define una tabla.
tr <tr> </tr> Define una fila en una tabla.
td <td> </td> Define una celda en una fila.
th <th> </th> Define una celda de encabezado en una fila.

Ejemplo de tabla:

Nombre Edad
Juan 20
María 22

3. Etiquetas de Listas

Nombre Inicio Cierre Descripción
ul <ul> </ul> Lista no ordenada.
ol <ol> </ol> Lista ordenada.
li <li> </li> Elemento de lista.

Ejemplo de lista no ordenada:

  • HTML
  • CSS
  • JavaScript

Ejemplo de lista ordenada:

  1. Inicio
  2. Desarrollo
  3. Conclusión

4. Etiquetas de Formularios

Nombre Inicio Cierre Descripción
form <form> </form> Define un formulario HTML.
input <input> No tiene Campo de entrada de datos.
label <label> </label> Etiqueta descriptiva para un campo.
select <select> </select> Menú desplegable.
option <option> </option> Opción dentro de un menú desplegable.
